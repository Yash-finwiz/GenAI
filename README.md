The FastAPI REST Server is implemented in the main.py file. The React code is in the app folder.

## How to run

1. Install the dependencies

```bash
pip3 install -r requirements.txt
```

2. Start the FastAPI server

```bash
python3 main.py
```

3. Start the React app

```bash
cd app
npm install
npm start
```
